# Azure Administrator Projects

Welcome to my Azure Administrator project portfolio. This repository showcases hands-on Azure projects demonstrating my skills in compute, networking, storage, security, monitoring, automation, identity, and containers.

## Projects

### 01. Azure Virtual Machine Deployment
Deploy a Windows Virtual Machine and configure secure remote access using RDP.

### 02. Azure Virtual Network (VNet) Peering
Configure VNet Peering to enable secure communication between two Azure Virtual Networks.

### 03. Azure Load Balancer with Two Virtual Machines
Configure an Azure Load Balancer to distribute traffic across two Virtual Machines.

### 04. Azure Network Security Group (NSG) Inbound Rules
Create and configure inbound security rules to protect Azure Virtual Machines.

### 05. Azure Key Vault with Managed Identity
Securely store secrets and enable applications to access them using Managed Identity.

### 06. Azure Blob Storage
Create and manage Azure Blob Storage for storing and accessing unstructured data.

### 07. Azure VM OS Disk Snapshot (Disaster Recovery)
Create and restore VM OS disk snapshots for backup and disaster recovery.

### 08. Azure Monitor with Alerts
Monitor Azure resources and configure alert rules for proactive monitoring.

### 09. Azure Virtual Machine Auto Shutdown
Configure automatic VM shutdown to optimize Azure resource usage and reduce costs.

### 10. Azure CLI Deployment
Deploy and manage Azure resources using Azure Command-Line Interface (CLI).

### 11. Azure Entra ID – User & Group Management with RBAC
Manage users, groups, and Role-Based Access Control (RBAC) for secure resource access.

### 12. Azure Container Instances
Deploy and manage containerized applications using Azure Container Instances (ACI).
