Azure Virtual Machine Deployment

## Objective
Deploy a Windows Virtual Machine in Azure and configure secure remote access.

## Services Used
- Azure Virtual Machine
- Virtual Network (VNet)
- Network Security Group (NSG)
- Public IP Address
- Resource Group

## Steps
1. Create a Resource Group
2. Create a Virtual Network and Subnet
3. Create a Windows Virtual Machine
4. Assign a Public IP Address
5. Configure NSG to allow RDP (Port 3389)
6. Connect to the Virtual Machine using Remote Desktop (RDP)
7. Verify successful login and VM functionality

## Outcome
Successfully deployed a Windows Virtual Machine, configured secure remote access, and verified connectivity using Remote Desktop.
