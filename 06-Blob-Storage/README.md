# Azure Blob Storage

## Objective
Create and configure Azure Blob Storage to securely store, manage, and access unstructured data such as files, images, and documents.

## Services Used
- Azure Storage Account
- Azure Blob Storage
- Blob Container
- Azure Portal
- Resource Group

## Prerequisites
- Active Azure Subscription
- Azure Portal Access
- Basic understanding of Azure Storage

## Steps
1. Create a Resource Group.
2. Create an Azure Storage Account.
3. Configure the storage account settings.
4. Create a Blob Container.
5. Upload sample files to the Blob Container.
6. Configure the appropriate access level for the container.
7. Access and verify the uploaded files.
8. Delete the uploaded files and container if no longer required.

## Outcome
Successfully created and configured Azure Blob Storage, uploaded files to a Blob Container, managed access permissions, and verified secure storage and retrieval of data.
