# Azure Virtual Machine Auto Shutdown

## Objective
Configure Auto Shutdown for an Azure Virtual Machine to automatically stop the VM at a scheduled time, reducing unnecessary resource usage and operational costs.

## Services Used
- Azure Virtual Machine
- Auto Shutdown
- Azure Portal
- Email Notifications
- Resource Group

## Prerequisites
- Active Azure Subscription
- Azure Portal Access
- Basic understanding of Azure Virtual Machines

## Steps
1. Create a Resource Group.
2. Deploy a Windows Virtual Machine.
3. Open the Virtual Machine from the Azure Portal.
4. Navigate to the *Auto-shutdown* settings.
5. Enable Auto Shutdown.
6. Set the shutdown time and select the appropriate time zone.
7. Configure email notifications (optional).
8. Save the Auto Shutdown configuration.
9. Wait for the scheduled shutdown time.
10. Verify that the Virtual Machine shuts down automatically as configured.

## Outcome
Successfully configured Auto Shutdown for an Azure Virtual Machine, ensuring automatic shutdown at the scheduled time to optimize resource utilization and reduce Azure costs.
