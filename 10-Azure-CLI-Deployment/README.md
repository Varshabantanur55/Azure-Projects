# Azure CLI Deployment

## Objective
Deploy and manage Azure resources using Azure Command-Line Interface (Azure CLI) to automate resource creation and administration tasks.

## Services Used
- Azure CLI
- Azure Virtual Machine
- Resource Group
- Virtual Network (VNet)
- Network Security Group (NSG)

## Prerequisites
- Active Azure Subscription
- Azure CLI Installed
- Basic understanding of Azure CLI commands

## Steps
1. Install Azure CLI on the local machine.
2. Sign in to Azure using the az login command.
3. Create a Resource Group using Azure CLI.
4. Create a Virtual Network and Subnet.
5. Create a Network Security Group (NSG).
6. Deploy a Windows Virtual Machine using Azure CLI commands.
7. Verify that all resources are created successfully.
8. Connect to the Virtual Machine using Remote Desktop (RDP).
9. List and verify the deployed resources using Azure CLI commands.
10. Delete the Resource Group using Azure CLI to clean up resources.

## Outcome
Successfully deployed and managed Azure resources using Azure CLI, demonstrating automation of infrastructure deployment and efficient resource management
